# Peak Load Forecasting
This is a project to forecast peak power load of Taiwan Power Company.<br>
Duration : Apr 2 to Apr 8, 2019
## Environment Setup
* Module installation
    ```
    $ pip install -r requirements.txt
    ```
* Execution ( python version 3.6 is recommended )
    ```
    $ python3 app.py
    ```
